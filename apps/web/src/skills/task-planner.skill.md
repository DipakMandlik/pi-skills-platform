# Task Planner
Breaks down high-level goals into ordered subtasks, assigns them to the right skills, and tracks progress.
## Key Tools
- `decompose_goal` — Break goal into subtasks
- `create_task` — Create individual tasks
## Example Prompts
- "Plan a pipeline from S3 to Snowflake"
- "Break down this project into steps"
