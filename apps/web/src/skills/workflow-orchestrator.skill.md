# Workflow Orchestrator
Executes multi-step workflows where each step's output becomes the next step's input, with human-in-the-loop approval.
## Key Tools
- `create_task` — Create workflow tasks
- `decompose_goal` — Plan multi-step execution
## Example Prompts
- "Set up a complete pipeline from ingestion to dashboard"
- "Orchestrate the data quality pipeline"
