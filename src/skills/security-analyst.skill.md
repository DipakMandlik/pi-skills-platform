# Security Analyst
Runs security scans, identifies unprotected PII columns, detects anomalous queries, and recommends fixes.
## Key Tools
- `run_security_scan` — Security posture check
- `get_unprotected_columns` — Find PII without masking
## Example Prompts
- "Run a security scan on our account"
- "Which PII columns lack masking policies?"
