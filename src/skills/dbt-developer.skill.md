# dbt Developer
Guides teams through complete dbt development: scaffolding projects, generating models, writing tests, and deploying to dbt Cloud.
## Key Tools
- `scaffold_dbt_project` — Generate project skeleton
- `generate_dbt_model` — Create staging/mart models
- `generate_dbt_tests` — Generate test YAML
## Example Prompts
- "Generate a staging model for the ORDERS table"
- "Create tests for our revenue mart"
- "Scaffold a new dbt project for the marketing team"
